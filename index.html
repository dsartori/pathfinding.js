Pathfinding Demo

This is an interactive demonstration of basic AI pathfinding techniques. Set up the maze, choose an algorithm and watch how the program finds a path from its starting point to the goal position.

Controls

Select the search algorithm to use from the dropdown box. is the starting location of the search, represents an impassible wall and is the goal.

Before starting the simulation, clicking on a position will toggle between open space and wall. Make sure the goal is reachable! When you're satisfied with your maze click the button to watch the search. Use the button to reset the simulation.

More Information

For more information on the algorithms used here, consult the wikipedia articles:

  • Breadth-first search
  • Depth-first search
  • A*
